## 💡 الفلسفة الأساسية

**"المحادثة + السياق في نفس الشاشة"**

الموظف لما يفتح محادثة، يشوف في نفس الوقت:
- تاريخ العميل الكامل
- فواتيره المعلّقة
- أوامر شغله
- ملاحظات الموظفين السابقين

بدون ما يفتح برنامج تاني أو يسأل حد تاني.

## 🌟 الفيتشرز الأساسية (MVP)

### الأعمدة الثلاثة:

#### العمود 1 — قائمة المحادثات:
- [ ] بحث فوري (اسم / رقم / آخر رسالة)
- [ ] فلاتر: الكل / مفتوحة / انتظار / مغلقة
- [ ] كل محادثة: avatar بإنشيالز + platform dot + اسم + وقت + preview
- [ ] Unread badge أحمر
- [ ] Status dot (أخضر/برتقالي/رمادي)
- [ ] فلتر الموظف المعيّن (للأدمن فقط)
- [ ] Queue count (غير معيّنة)

#### العمود 2 — المحادثة:
- [ ] Header: اسم + platform pill + حالة + أزرار actions
- [ ] Messages: في/خارج مع وقت وحالة
- [ ] ميديا: صور/فيديو/صوت/ملفات inline
- [ ] Internal Notes (تظهر باللون الأصفر — مش بتتبعت)
- [ ] Reply box:
  - Templates بـ "/" 
  - AI suggestions
  - إرفاق ملف
  - رسالة صوتية
  - Emoji picker
  - Shift+Enter = سطر جديد / Enter = إرسال

#### العمود 3 — سياق العميل:
- [ ] بيانات العميل (اسم / تليفون / إيميل / مدينة)
- [ ] CRM status badge (Lead/Client/VIP)
- [ ] آخر 3 فواتير (رقم / مبلغ / حالة)
- [ ] الذمم المستحقة مع زر "سجّل دفعة"
- [ ] آخر 3 أوامر شغل
- [ ] أزرار سريعة: فاتورة جديدة / أمر شغل / بعت PDF / إضافة كعميل

## ⚡ الفيتشرز التي تتفوق على المنافسين

### 1. Customer Context Panel (الأقوى)
مش موجود بنفس العمق في أي منافس — الفواتير + الأوردرات + الذمم في نفس شاشة الرد.

### 2. Send Invoice in One Click
من داخل المحادثة مباشرة — اختار فاتورة → تتبعت مع رابط دفع تلقائي.

### 3. Work Order Tracking
تتبع أمر الشغل من الاستلام للتسليم من داخل المحادثة.

### 4. Internal Notes
موظف يكتب ملاحظة للموظف اللي هيرد بعده — مش بتتبعت للعميل.

### 5. Inbox-Specific Permissions
صلاحيات منفصلة تماماً عن صلاحيات النظام — inbox agents لهم controls خاصة.

### 6. Arabic-First Design
RTL native، font Cairo، كل شيء مفكّر للمستخدم العربي.

## 🎨 Design System

### الألوان:
```css
--brand:      #1B5E30  /* أخضر أريج */
--bg:         #F8FAFC  /* خلفية عامة */
--surface:    #FFFFFF  /* بطاقات */
--border:     #E2E8F0  /* حدود */
--text-main:  #111827  /* نص أساسي */
--text-sub:   #6B7280  /* نص ثانوي */
--unread:     #EF4444  /* badge غير مقروء */
--waiting:    #F59E0B  /* انتظار */
--closed:     #9CA3AF  /* مغلق */
```

### Platform Colors:
```css
--tg:   #229ED9  /* Telegram */
--wa:   #25D366  /* WhatsApp */
--fb:   #1877F2  /* Messenger */
--ig:   #E1306C  /* Instagram */
```

### Typography:
- Font: Cairo (Arabic-optimized)
- Sizes: 11px (meta) / 12px (preview) / 13px (body) / 15px (name) / 18px (header)
