📱 واتساب Business API

📌 كيف تعرف إن رقمك هو "WhatsApp Business API"؟
لو رقمك مربوط بـ Respond.io أو Wati أو 360dialog أو أي منصة messaging — إذاً هو رقم API ✅
لو الرقم عليه تطبيق WhatsApp شخصي أو WhatsApp Business العادي — إذاً مش API ❌ (استخدم الزر التاني)
⚠️ الخطوة الأولى: افصل الرقم من المنصة القديمة (Respond / Wati / غيرها)
① افتح business.facebook.com/settings
② من القائمة: Accounts → WhatsApp Accounts
③ اختار حساب الواتساب بتاعك → اضغط Settings
④ من القائمة اليسرى اضغط WhatsApp Manager → ثم Phone Numbers
⑤ اضغط على الرقم → ثم Settings (أيقونة الترس)
⑥ دوّر على "Assigned Partners" أو "Connected App" → اضغط Remove / Revoke
⑦ انتظر 2-3 دقائق → الرقم بقى حر ✅
🎯 خطوات الربط — 3 خطوات فقط
1
جيب الـ Phone Number ID
① افتح business.facebook.com/settings
② من القائمة: Accounts → WhatsApp Accounts
③ اضغط على حساب الواتساب بتاعك
④ من القائمة اليسرى اضغط WhatsApp Manager → ثم Phone Numbers
⑤ هتشوف جدول برقمك — انسخ الـ Phone Number ID (رقم طويل زي: 123456789012345)
🔗 افتح Meta Business Settings ↗
2
أنشئ Token دائم (System User Token)
① افتح business.facebook.com/settings → System Users
② اضغط Add → أدخل أي اسم (مثلاً: Areej Bot) → النوع: Employee → اضغط Create
③ بعد الإنشاء اضغط على المستخدم الجديد → اضغط Add Assets
④ اختار WhatsApp Accounts من القائمة → اختار حسابك → فعّل Manage WhatsApp Business → اضغط Save
⑤ ارجع للـ System User → اضغط Generate New Token
⑥ اختار أي تطبيق من القائمة (لو ما عندكش تطبيق، اضغط أنشئ واحد بسيط من هنا ← مجاني وبيأخذ دقيقتين)
⑦ فعّل whatsapp_business_messaging + whatsapp_business_management
⑧ اضغط Generate Token → انسخه فوراً واحفظه — مش هيتعرضلك تاني أبداً!
🔗 افتح System Users ↗
3
اضبط الـ Webhook في Meta (مرة واحدة بس)
① افتح developers.facebook.com/apps → اختار تطبيقك
② من القائمة اليسرى: WhatsApp → Configuration
③ اضغط Edit في قسم Webhook
④ في Callback URL: انسخ الـ URL المكتوب في الأسفل وضعه هنا
⑤ في Verify token: انسخ الـ Verify Token المكتوب في الأسفل وضعه هنا
⑥ اضغط Verify and Save — هيقولك ✅ Verified
⑦ اضغط Manage بجوار Webhook Subscriptions → فعّل messages فقط
📌 متى تختار هذا الخيار؟
لو عندك رقم هاتف مش مستخدم في أي واتساب، وعايز تربطه بـ WhatsApp Business API من الصفر ✅
⚠️ الرقم ده لازم يكون مش عليه واتساب خالص — الرقم اللي عليه واتساب شخصي أو بيزنس مش هينفع مباشرة (لازم تحذف منه الواتساب الأول)
📋 اللي محتاجه قبل ما تبدأ
✅ حساب Meta Business Manager — سجّل من business.facebook.com لو مش عندك
✅ رقم هاتف غير مستخدم في أي واتساب نهائياً (شخصي أو بيزنس)
✅ بطاقة بنكية (Visa / Mastercard) مربوطة بحساب Meta للفوترة
✅ متصفح Chrome أو Edge (مش Firefox)
🪜 خطوات إنشاء رقم جديد من الصفر
1
أنشئ تطبيق على Meta Developers
① افتح developers.facebook.com/apps/create
② اختار Other → اضغط Next → اختار Business → اضغط Next
③ أدخل اسم (مثلاً: Areej WhatsApp) → اختار Business Account → اضغط Create App
🔗 إنشاء تطبيق جديد ↗
2
أضف منتج WhatsApp للتطبيق
① بعد إنشاء التطبيق هتشوف صفحة "Add products to your app"
② ابحث عن WhatsApp في القائمة → اضغط Set up
③ اختار الـ Meta Business Account بتاعك → اضغط Continue
④ هتتوجّه لصفحة WhatsApp → Getting Started تلقائياً
3
أضف رقم الهاتف
① في صفحة Getting Started → اضغط Add phone number
② أدخل اسم شركتك → اختار تصنيف النشاط
③ أدخل رقم الهاتف → اختار طريقة التحقق (SMS أو مكالمة صوتية)
④ أدخل الكود الذي وصلك → اضغط Next
⑤ انتظر حتى يظهر الرقم بحالة Connected ✅
4
جيب الـ Phone Number ID والـ WABA ID
① في نفس الصفحة (WhatsApp → API Setup)
② بجوار رقمك هتشوف Phone number ID — انسخه
③ فوقه هتشوف WhatsApp Business Account ID — انسخه كمان
5
أنشئ Token دائم (System User Token)
① افتح business.facebook.com/settings → System Users
② اضغط Add → أدخل اسم (مثلاً: Areej Bot) → النوع: Employee → Create
③ اضغط على المستخدم الجديد → اضغط Add Assets
④ اختار WhatsApp Accounts → اختار حسابك → فعّل Manage WhatsApp Business → Save
⑤ ارجع للـ System User → اضغط Generate New Token
⑥ اختار التطبيق اللي أنشأته للتو
⑦ فعّل: whatsapp_business_messaging + whatsapp_business_management
⑧ اضغط Generate Token → انسخه فوراً واحفظه — مش هيتعرضلك تاني!
🔗 System Users ↗
6
اضبط الـ Webhook
① في Meta Developers → تطبيقك → من القائمة اليسرى: WhatsApp → Configuration
② اضغط Edit في قسم Webhook
③ في Callback URL: انسخ الـ URL المكتوب في الأسفل وضعه هنا
④ في Verify token: انسخ الـ Verify Token المكتوب في الأسفل وضعه هنا
⑤ اضغط Verify and Save — لازم يقولك ✅ Verified
⑥ اضغط Manage بجوار Webhook Subscriptions → فعّل messages فقط → Save

⚙️ أدخل البيانات هنا

رقم طويل من Meta Developers → WhatsApp → API Setup
من نفس الصفحة فوق الـ Phone Number ID
⚠️ لازم يكون System User Token دائم — مش الـ Token المؤقت اللي صالح 24 ساعة بس
📝 الـ Templates بتاعتك
اضغط "تحديث" لعرض الـ Templates

➕ Template جديد

حروف إنجليزية صغيرة + أرقام + _ فقط — مثال: order_confirmation
استخدم {{1}} {{2}} للمتغيرات — كل رقم = متغير لازم تملأه لما تبعت الرسالة
👁 معاينة (WhatsApp bubble)
📊 تحليلات الواتساب
اضغط "تحديث" لعرض التحليلات
تقسيم حسب نوع المحادثة
💰 تكلفة تقديرية للشهر الجاري
تقديري بناءً على أسعار Meta لمصر (EG) — شوف الأسعار
📊 إحصائيات محلية (من بيانات Areej Pro)

فيسبوك ماسنجر

⚠️ تحتاج موافقة Meta
ربط Messenger يحتاج إنك تعمل Meta App وتحصل على Page Access Token. لو مش عندك خبرة بـ Meta Developers، اتبع الخطوات دي بدقة.
🔌 خطوات الربط
1
أنشئ Meta App
افتح Meta for Developers → Create App → Business → اختار "Messenger"
2
اربط صفحة فيسبوك
من داشبورد الـ App → Messenger → Settings → Subscribed Pages → اربط صفحتك
3
احصل على Page Access Token
Messenger Settings → Access Tokens → Generate Token للصفحة المطلوبة
4
اضبط الـ Webhook
Webhooks → Edit → ارفع الـ Webhook URL التالي + Verify Token → اشترك في messages, messaging_postbacks
5
احفظ الإعدادات وابدأ
أدخل الـ Page Access Token + Page ID في الأسفل واضغط حفظ
Token طويل يبدأ بـ EAAB
من صفحة About في صفحتك على فيسبوك

إنستجرام

💡 نفس الـ Meta App
لو سبق وعملت Meta App لـ Messenger، ممكن تستخدم نفسها هنا. بس محتاج تضيف صلاحية instagram_basic وinstagram_manage_messages
🔌 خطوات الربط
1
حوّل حسابك لـ Professional
Instagram Settings → Account → Switch to Professional Account → Business
2
اربط بصفحة فيسبوك
Instagram Settings → Linked Accounts → Facebook → اربط صفحتك
3
أضف Instagram في Meta App
Meta Developers → App Products → Add Instagram → اربط الحساب
4
اضبط الـ Webhook
Webhooks → Add Subscription → ارفع الـ URL التالي → اشترك في messages, messaging_handovers
5
احفظ الإعدادات
أدخل الـ Instagram Page Token + Account ID في الأسفل واضغط حفظ
من الـ Meta App → Instagram → Account ID

📱 واتساب QR Code

⚠️ مهم — اقرأ قبل البدء
  • هذا الخيار غير رسمي — يعمل عبر محاكاة واتساب Web
  • قد يُحظر الرقم من واتساب إذا تم الاستخدام بشكل مكثف
  • لا يدعم رسائل Template المدفوعة
  • مناسب للـ تجربة والاختبار فقط — للإنتاج استخدم واتساب API الرسمي
💡 كيف يعمل؟
يعتمد على مكتبة whatsapp-web.js التي تشغّل نسخة مخفية من واتساب Web على الخادم. تربطها برقمك عبر مسح QR Code مرة واحدة، وبعدها الخادم يستقبل ويرسل الرسائل نيابةً عنك.

🪜 خطوات الربط

1
اضغط "تشغيل خدمة QR" أدناه
النظام سيبدأ تلقائياً في إنشاء جلسة واتساب وتوليد الـ QR Code — قد يستغرق 15-30 ثانية.
2
امسح الـ QR Code من هاتفك
افتح تطبيق واتساب على هاتفك:
⋯ القائمة → الأجهزة المرتبطة → ربط جهاز → امسح الـ QR
3
انتظر ظهور "متصل ✅"
بعد المسح بثوانٍ ستظهر رسالة نجاح ويبدأ الـ inbox في استقبال رسائلك.
📱
الخدمة غير مشغّلة
اضغط "تشغيل خدمة QR" للبدء
